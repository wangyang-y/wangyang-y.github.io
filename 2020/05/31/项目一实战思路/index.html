项目一实战总结

会员统计

设计年龄段，0-6岁为婴儿，7-12岁为少儿，13-17岁为青少年，18-45岁为青年、46-69岁为中年、>69岁为老年，为达到一目了然各个年龄段的人数高低，选择柱状图展示。

心得

使用ECharts很容易达到效果，使用时我们不需要关系样式怎么设置，直接去ECharts官网找到合适的案例，粘贴到页面上即可。重要的是根据案例中的数据，构造出合适的数据模型动态的赋值。

前端调用实现

// 基于准备好的dom，初始化echarts实例
        var myChart1 = echarts.init(document.getElementById('chart1'));

        axios.get("/report/getMemberAgeReport.do").then((res)=>{
            // 指定图表的配置项和数据
            myChart1.setOption(
                {
                    title: {
                        text: '会员年龄分布统计'
                    },
                    tooltip: {},
                    legend: {
                        data:['年龄段人数']
                    },
                    xAxis: {
                        data: res.data.data.ageStages
                    },
                    yAxis: {},
                    series: [{
                        name: '年龄段人数',
                        type: 'bar',
                        data: res.data.data.ageCountNum
                    }]
                }
                );
        });

后台逻辑

@RequestMapping("/getMemberAgeReport")
    public Result getMemberAgeReport() {
        List<Map<String,Object>> ageStagesMapList = memberService.findMemberCountByBirthdy();
        Map<String, Object> data = new HashMap<>();
        List<String> ageStages = new ArrayList<>();
       
        //纵坐标数据,统计每个年龄段的人数
        List<Long> ageCountNum = new ArrayList<>();
        if(ageStagesMapList != null){
            for (Map<String, Object> stringObjectMap : ageStagesMapList) {
                //拿到map集合{name=青少年, num=12}
                String str = (String) stringObjectMap.get("name");
                //年龄段表示字段 x
                ageStages.add(str);
                Long value = (Long) stringObjectMap.get("num");
                //年龄段的值 y
                ageCountNum.add(value);
            }
            //将横纵坐标的list 接入map 集合
            data.put("ageStages", ageStages);
            data.put("ageCountNum",ageCountNum);
            return new Result(true, "查询成功", data);
        }
        return new Result(false, "查询失败");
    }
#### SQL
<select id="findMemberCountByBirthdy" resultType="map">
        SELECT t.name,COUNT(1) as num FROM (SELECT  CASE
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)> 0 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<  6  THEN '婴幼儿'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=7 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<   12  THEN '少儿'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=13 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<   17  THEN '青少年'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=18 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<   45  THEN '青年'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=46 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<   69  THEN '中年'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=79 AND ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)<   89  THEN '老年'
        WHEN  ROUND(DATEDIFF(CURDATE(), birthday)/365.2422)>=90 THEN '90以上'
        ELSE '未知' END as name FROM t_member) as t GROUP BY t.name;
    </select>

SQL分析

年龄采取四舍五入的方式，如出生日期2018-08-02，如果今天是2018-09-01差一天满，将处理为1岁显示。

如果不需要四舍五入则使用： TIMESTAMPDIFF(YEAR, @birthday, CURDATE())

round(): 函数 用于四舍五入。

DATEDIFF()：返回两个日期之间的天数。

CURDATE() 函数返回当前的日期。

注意大于号和小于号在mybatis 中转义，不转义会和标签的 <>混淆。
